AI Python for Beginners

Course Overview

AI Python for Beginners is a comprehensive course taught by renowned AI leader Prof Andrew Ng. This course is designed to introduce students to the fundamentals of Python programming with a focus on its applications in Artificial Intelligence.

What You'll Learn

  • Python basics: variables, data types, and control structures
  • Functions and modules in Python
  • Introduction to AI concepts and terminologies
  • Basic machine learning algorithms using Python
  • Practical AI projects and applications

Course Duration

8 weeks, with approximately 5-7 hours of study per week
